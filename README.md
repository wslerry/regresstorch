# regresstorch

Regresstorch is a regression model applied into PyTorch framework  on a CUDA-capable NVIDIA GPU

## How to use?

```shell
python train.py [--epoch EPOCH] [--batch-size BATCH_SIZE] [--input INPUT] [--var1 VAR1] [--var2 VAR2] [--adam] [--device DEVICE]
```
